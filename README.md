# java-cli-maven-graphql-hello

## Description
A POC for a maven graphql project.

## Tech stack
- java
- maven

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
